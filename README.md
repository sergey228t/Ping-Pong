# Ping-Pong

Пинг-Понг - это мультиплеерная игра реализованная на языке программирования Pyton. Игрок слева(1) управляет левой ракеткой WASD'ом, а правый(1) правой с помощью стрелочек.

Ping-Pong is a multiplayer game implemented in the Python programming language. The player on the left (1) controls the left racket with WASD, and the player on the right (1) controls the right racket with the arrow keys.
